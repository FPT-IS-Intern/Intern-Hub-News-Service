# 04 - API Module

Endpoints:
- `POST /news`
- `GET /news`
- `GET /news/{id}`
- `PUT /news/{id}`
- `DELETE /news/{id}`
- `GET /news/health`

Main classes:
- `NewsController`
- `HealthCheckController`
- request DTOs: `CreateNewsRequest`, `UpdateNewsRequest`
- response DTO: `NewsResponse`
