# 08 - Naming Conventions

Conventions used in this base:
- domain model: `NewsModel`
- port interface: `NewsRepository`
- use case interface/impl: `NewsUsecase`, `NewsUsecaseImpl`
- JPA entity: `News`
- JPA repo: `NewsJpaRepository`
- API controller: `NewsController`
- DTOs: `CreateNewsRequest`, `UpdateNewsRequest`, `NewsResponse`
